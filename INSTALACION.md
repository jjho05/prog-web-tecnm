# Guía de Instalación

## Requisitos del Sistema

- **Sistema Operativo:** Windows, macOS o Linux
- **Python:** Versión 3.8 o superior
- **Navegador:** Chrome, Firefox, Safari o Edge actualizado

## Pasos de Instalación

1. Clona este repositorio:
   ```bash
   git clone https://github.com/jjho05/prog-web-tecnm.git
   ```

2. Instala las dependencias (si aplica para scripts de backend):
   ```bash
   pip install -r requirements.txt
   ```

3. Navega a los ejemplos de código:
   ```bash
   cd content/unidad1/codigos/
   ```

# Programación Web (AEB-1055)

Skill educativo completo para la materia de Programación Web del Tecnológico Nacional de México.

## Contenido

Este skill cubre el 100% del programa oficial mediante:
- Conceptos teóricos detallados
- Ejemplos de código funcionales
- Ejercicios prácticos
- Guías paso a paso

## Estructura

- `content/unidadX/`: Contenido teórico desglosado por unidad
- `content/unidadX/codigos/`: Ejemplos prácticos ejecutables
- `references/`: Material de apoyo y temario completo

## Instrucciones para el Asistente

Al usar este skill:
1. Sigue estrictamente el orden del temario oficial.
2. Provee explicaciones claras y concisas en español.
3. Asegura que todo código proporcionado sea funcional y siga buenas prácticas.
4. Cita siempre la fuente oficial (Programa AEB-1055) en los materiales generados.

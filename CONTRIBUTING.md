# Cómo Contribuir

¡Gracias por tu interés en contribuir a Skills ISC TecNM! 

## Proceso de Contribución

1. Haz un Fork del repositorio.
2. Crea una rama para tu feature (`git checkout -b feature/NuevaFeature`).
3. Realiza tus cambios siguiendo los estándares.
4. Haz Commit de tus cambios (`git commit -m 'Agrega nueva feature'`).
5. Haz Push a la rama (`git push origin feature/NuevaFeature`).
6. Abre un Pull Request.

## Estándares

- Todo el contenido debe estar en español.
- El código debe estar comentado y probado.
- Seguir la estructura de carpetas definida.

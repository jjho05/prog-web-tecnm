# Unidad 2: Maquetado Web

## Descripción

Esta unidad se enfoca en la estructura y presentación de las páginas web. Aprenderás a crear documentos semánticos con HTML5 y a aplicar estilos profesionales con CSS3.

## Contenido

### [2.1 Introducción a HTML](2.1.md)
- Historia resumida de la web.
- DOM y Estándares.

### [2.2 Estructura Global](2.2.md)
- El esqueleto de una página (`html`, `head`, `body`).
- Metadatos para redes sociales y buscadores.

### [2.3 Elementos Básicos](2.3.md)
- Texto, Enlaces y Listas.
- Multimedia (Audio, Video, Imágenes).
- Etiquetas semánticas (`header`, `nav`, `footer`).

### [2.4 Formularios](2.4.md)
- Cómo pedir datos al usuario.
- Tipos de inputs y validación automática.

### [2.5 Introducción a CSS](2.5.md)
- Separación de Estructura (HTML) y Estilo (CSS).
- Unidades de medida (`px`, `rem`, `%`).

### [2.6 Selectores CSS](2.6.md)
- Cómo apuntar a los elementos exactos.
- ID vs Clase vs Etiqueta.
- Especificidad (¿Quién gana?).

### [2.7 Modelo de Caja y Layouts](2.7.md)
- Padding, Margin y Border.
- `box-sizing: border-box`.
- Flexbox y Grid (Diseño moderno).

## Objetivos de Aprendizaje

Al completar esta unidad, serás capaz de:
- ✅ Crear páginas web semánticas con HTML5.
- ✅ Diseñar formularios interactivos y accesibles.
- ✅ Aplicar estilos CSS3 para diseño visual.
- ✅ Dominar Flexbox y CSS Grid para layouts responsivos.

## Recursos Adicionales

- **Herramientas:** VS Code, Chrome DevTools.
- **Práctica:** Flexbox Froggy, Grid Garden.
- **Tiempo estimado:** 10-12 horas.

# Unidad 4: Programación del Lado del Servidor

## Descripción

Esta unidad traslada el desarrollo al servidor. Aprenderás a construir APIs robustas usando Node.js, Express y bases de datos, entendiendo la arquitectura del backend moderno.

## Contenido

### [4.1 Introducción a Node.js](4.1.md)
- Runtime vs Lenguaje
- Single Threaded & Event Loop
- Sistema de Módulos (CJS vs ESM)

### [4.2 Estructuras de Control (Lógica)](4.2.md)
- Patrón Middleware
- Control de flujo asíncrono
- Manejo de Errores centralizado

### [4.3 Tratamiento de Formularios](4.3.md)
- Body Parsing (JSON vs URL)
- Validación con Zod
- Seguridad (CSRF)

### [4.4 Objetos del Servidor](4.4.md)
- `req` y `res` a fondo
- Headers y Cookies
- JWT y Autenticación Stateless

### [4.5 Arquitectura (Clases)](4.5.md)
- Capas: Controller, Service, Repository
- Inyección de Dependencias
- Principios SOLID en Backend

### [4.6 Acceso a Datos (DB)](4.6.md)
- SQL vs NoSQL
- ORMs (Prisma) y Migraciones
- Optimización (Pools, N+1)

## Objetivos de Aprendizaje

Al completar esta unidad, serás capaz de:
- ✅ Configurar un servidor web con Node.js y Express.
- ✅ Diseñar APIs RESTful profesionales.
- ✅ Conectar tu aplicación a una Base de Datos.
- ✅ Implementar la arquitectura MVC en el Backend.

## Recursos Adicionales

- **Docs:** Expressjs.com, Nodejs.org.
- **Herramientas:** Postman, Thunder Client.
- **Tiempo estimado:** 15-20 horas.

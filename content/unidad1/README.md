# Unidad 1: Antecedentes

## Descripción

Esta unidad cubre la evolución histórica y tecnológica de la programación web, desde los inicios de Internet hasta la arquitectura Cliente-Servidor moderna. Establece las bases conceptuales necesarias para el desarrollo web.

## Contenido

### [1.1 Evolución de las aplicaciones web](1.1.md)
- Historia de Internet y la WWW
- Web 1.0, 2.0, 3.0 y 4.0
- Evolución de los navegadores y servidores

### [1.2 Arquitectura de las aplicaciones web](1.2.md)
- Modelo Cliente-Servidor
- Arquitectura de 2, 3 y N capas
- MVC (Modelo-Vista-Controlador)

### [1.3 Tecnologías para el desarrollo de aplicaciones web](1.3.md)
- Frontend vs Backend
- Lenguajes (HTML, CSS, JS, PHP, Java, Python)
- Frameworks y Librerías

### [1.4 Planificación de aplicaciones web](1.4.md)
- Ciclo de vida del desarrollo web
- Requerimientos funcionales y no funcionales
- Wireframes y Prototipado

## Objetivos de Aprendizaje

Al completar esta unidad, serás capaz de:
- ✅ Comprender la evolución histórica de la Web.
- ✅ Describir la arquitectura Cliente-Servidor.
- ✅ Identificar las tecnologías clave del desarrollo web.
- ✅ Planificar un proyecto web desde cero.

## Recursos Adicionales

- **Libros:** "Learning Web Design" de Jennifer Robbins.
- **Documentación:** MDN Web Docs.
- **Tiempo estimado:** 6-8 horas.

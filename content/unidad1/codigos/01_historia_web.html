Historia de la Web

1989 - Web 1.0

El Inicio

Tim Berners-Lee inventa la WWW. Páginas estáticas, solo lectura. HTML básico.

2004 - Web 2.0

La Web Social

Nace Facebook, YouTube. El usuario genera contenido. AJAX permite interactividad sin recargar.

2010 - Web 3.0

La Web Semántica

Uso de metadatos, IA básica, personalización y movilidad (Smartphones First).

2020+ - Web 4.0

La Web Simbiótica

IA generativa, asistentes virtuales, IoT y conexión total.

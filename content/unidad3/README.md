# Unidad 3: Programación del Lado del Cliente

## Descripción

Esta unidad introduce la lógica de programación en el navegador. Aprenderás JavaScript moderno (ES6+) para crear interactividad, manipular el DOM y consumir APIs.

## Contenido

### [3.1 Introducción al lenguaje](3.1.md)
- Historia de JavaScript y ES6+
- Variables (`var`, `let`, `const`) y Tipos de Datos
- Operadores y Funciones (Arrow Functions)

### [3.2 Manejo de Frameworks (Vue.js / React)](3.2.md)
- ¿Por qué usar un Framework?
- Componentes y Reactividad
- Virtual DOM

### [3.3 Estructuras de control](3.3.md)
- Condicionales Modernos (`??`, `?.`)
- Iteración Funcional (`map`, `filter`)
- **Async/Await** y Promesas

### [3.4 Manipulación de Objetos y DOM](3.4.md)
- DOM API moderna (`querySelector`)
- Eventos y Delegación
- Fetch API (Consumo de datos)
- `localStorage` y JSON

## Objetivos de Aprendizaje

Al completar esta unidad, serás capaz de:
- ✅ Escribir lógica compleja con JavaScript ES6+.
- ✅ Manipular el DOM dinámicamente.
- ✅ Consumir datos de APIs externas (Fetch).
- ✅ Entender los fundamentos de frameworks modernos.

## Recursos Adicionales

- **Sitios:** JavaScript.info, Eloquent JavaScript.
- **Tools:** Node.js (para ejecutar JS fuera del navegador).
- **Tiempo estimado:** 15-20 horas.
